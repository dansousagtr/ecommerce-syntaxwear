# Documentação do projeto

Este diretório contém notas e instruções relacionadas à preparação do projeto.

Favicons criados (placeholders):
- `assets/images/favicon.svg`
- `assets/images/favicon-32.svg`
- `assets/images/favicon-maskable.svg`

Como usar:
- Adicione links no `<head>` do `index.html` para os favicons, por exemplo:

```
<link rel="icon" href="assets/images/favicon.svg" />
<link rel="icon" sizes="32x32" href="assets/images/favicon-32.svg" />
<link rel="mask-icon" href="assets/images/favicon-maskable.svg" color="#0E7AFE" />
```

Substitua os SVGs por versões finais (.png/.ico/.webmanifest) conforme necessário.
